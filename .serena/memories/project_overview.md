Clawith is an open-source multi-agent collaboration platform with a FastAPI backend and React 19 + Vite frontend. Backend lives in backend/app with domain routers, SQLAlchemy async models, and services for agent execution, triggers, channels, governance, and approvals. Frontend lives in frontend/src with pages, shared components, Zustand stores, React Query fetching, and i18n via en.json and zh.json. All routers are mounted under /api. Version is tracked in root VERSION. Multi-tenancy is a core constraint: tenant-scoped entities and queries are required.
