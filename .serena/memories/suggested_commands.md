Setup: `bash setup.sh` or `bash setup.sh --dev`. Start services: `bash restart.sh`. Backend dev server: `cd backend && source .venv/bin/activate && uvicorn app.main:app --host 0.0.0.0 --port 8008 --reload`. Backend lint/format: `cd backend && ruff check app/ --fix && ruff format app/`. Backend tests: `cd backend && pytest` or `pytest tests/test_foo.py -v`. Frontend dev: `cd frontend && npm run dev`. Frontend build: `cd frontend && npm run build`. Docker full stack: `docker compose up -d` or `docker compose up -d --build`.
